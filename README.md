# PPG-Heart-Rate-Monitoring
## 通过对ppg信号数据进行处理，计算得到心率值

### 文件介绍
-PPG信号检测.ipynb为数据处理程序  
-心率检测参数实验情况.xlsx为不同参数选择的对比结果  

### 数据情况
使用智能手环采集3个多小时手腕PPG和加速度计数据，采样率均为128Hz，其中加速度计数据为3轴（X、Y、Z）数据。  

### 处理流程
-0.1Hz高通滤波去直流、去除极端基线漂移  
-结合加速度数据的NLMS自适应滤波器去除运动伪影  
-0.5~5Hz的巴特沃斯带通滤波  
-0.5Hz以上高通滤波去除基线漂移  
-移动平均滤波光滑数据  
-自适应峰值检测  
-计算心率
